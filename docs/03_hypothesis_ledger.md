## First Decision

The first serious analysis should not be a reading. It should be a positional grammar:

1. Normalize directionality.
2. Extract start, end, and medial sign distributions.
3. Split by artifact type and site.
4. Identify signs whose positional entropy is unusually low.
5. Use those signs to define candidate functional classes.
